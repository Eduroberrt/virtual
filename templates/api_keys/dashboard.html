API Documentation

Integrate SMS verification services into your application using our RESTful API. All requests use DaisySMS (USA numbers) as the backend provider.

Base URL: {{ request.scheme }}://{{ request.get_host }}/api/v1

Authentication

All API requests require authentication using an API key in the X-API-Key header:

X-API-Key: YOUR_API_KEY

Example using cURL:

curl -H "X-API-Key: your_32_character_api_key" \
     {{ request.scheme }}://{{ request.get_host }}/api/v1/balance

API Endpoints

GET /balance

Description: Get your current account balance

Response Example:
{
  "success": true,
  "balance": 5000.00,
  "currency": "NGN"
}

POST /purchase

Description: Purchase a phone number for SMS verification

Request Body (JSON):
Field Type Required Description
service string Yes Service code (e.g., "wa", "tg", "go")
max_price number No Maximum price in NGN you're willing to pay
area_codes array No Specific area codes (e.g., ["503", "202"]) - adds 20% premium
carriers array No Carriers: "tmo", "vz", "att" - adds 20% premium
number string No Specific 10-digit number - adds 20% premium
Example Request (Basic):
POST /api/v1/purchase/
Content-Type: application/json

{
  "service": "wa",
  "max_price": 2000.00
}
Example Request (With Premium Filters):
POST /api/v1/purchase/
Content-Type: application/json

{
  "service": "wa",
  "area_codes": ["503", "202"],
  "carriers": ["tmo"]
}
Response Example (Success):
{
  "success": true,
  "order_id": "api_1234567890abcdef",
  "rental_id": "12345",
  "phone_number": "15034567890",
  "service": "wa",
  "service_name": "WhatsApp",
  "price": 1980.00,
  "currency": "NGN",
  "status": "WAITING",
  "has_premium": true,
  "message": "Number rented successfully"
}
Premium Filters: Using area_codes, carriers, or specific number adds a 20% premium to the base price.

GET /order/{order_id}

Description: Check order status and retrieve SMS messages

Response Example (Waiting for SMS):
{
  "success": true,
  "order_id": "api_1234567890abcdef",
  "rental_id": "12345",
  "phone_number": "15034567890",
  "status": "WAITING",
  "service": "wa",
  "sms": []
}
Response Example (SMS Received):
{
  "success": true,
  "order_id": "api_1234567890abcdef",
  "rental_id": "12345",
  "phone_number": "15034567890",
  "status": "RECEIVED",
  "service": "wa",
  "sms": [
    {
      "code": "123456",
      "text": "Your WhatsApp code is 123456",
      "received_at": "2024-01-15T10:32:15Z"
    }
  ]
}
Status Values:
  • WAITING - Number rented, waiting for SMS
  • RECEIVED - SMS received successfully
  • DONE - Order completed
  • CANCELLED - Order cancelled
Polling: Poll this endpoint every 3-5 seconds to check for SMS codes. Don't poll more frequently to avoid rate limits.

POST /order/{order_id}/cancel

Description: Cancel an order and receive a refund

Response Example:
{
  "success": true,
  "status": "CANCELLED",
  "refund_amount": 1650.00
}

GET /orders

Description: Get list of your orders

Query Parameters:
Parameter Description
limit Number of orders (max 100, default 20)
status Filter by status (WAITING, RECEIVED, etc.)

Code Examples

Python
import requests

API_KEY = "your_api_key_here"
BASE_URL = "{{ request.scheme }}://{{ request.get_host }}/api/v1"

headers = {
    "X-API-Key": API_KEY,
    "Content-Type": "application/json"
}

# Purchase a number
response = requests.post(f"{BASE_URL}/purchase/", 
    headers=headers,
    json={"service": "wa"}
)
result = response.json()
order_id = result["order_id"]

# Check for SMS
import time
while True:
    response = requests.get(f"{BASE_URL}/order/{order_id}", headers=headers)
    result = response.json()
    
    if result["sms"]:
        code = result["sms"][0]["code"]
        print(f"SMS Code: {code}")
        break
    
    time.sleep(5)
JavaScript (Node.js)
const axios = require('axios');

const API_KEY = 'your_api_key_here';
const BASE_URL = '{{ request.scheme }}://{{ request.get_host }}/api/v1';

const headers = {
    'X-API-Key': API_KEY,
    'Content-Type': 'application/json'
};

// Purchase and wait for SMS
async function purchaseAndWaitForSMS() {
    const response = await axios.post(`${BASE_URL}/purchase/`, 
        { service: 'wa' },
        { headers }
    );
    const orderId = response.data.order_id;
    
    while (true) {
        const check = await axios.get(`${BASE_URL}/order/${orderId}`, { headers });
        
        if (check.data.sms.length > 0) {
            console.log('SMS Code:', check.data.sms[0].code);
            break;
        }
        
        await new Promise(resolve => setTimeout(resolve, 5000));
    }
}
PHP
<?php
$apiKey = 'your_api_key_here';
$baseUrl = '{{ request.scheme }}://{{ request.get_host }}/api/v1';

$headers = [
    'X-API-Key: ' . $apiKey,
    'Content-Type: application/json'
];

// Purchase a number
$ch = curl_init($baseUrl . '/purchase/');
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode(['service' => 'wa']));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($ch);
$result = json_decode($response, true);
$orderId = $result['order_id'];

// Check for SMS
while (true) {
    $ch = curl_init($baseUrl . '/order/' . $orderId);
    curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    $response = curl_exec($ch);
    $result = json_decode($response, true);
    
    if (count($result['sms']) > 0) {
        echo "SMS Code: " . $result['sms'][0]['code'] . "\n";
        break;
    }
    
    sleep(5);
}
?>
